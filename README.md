# Amazon-Sales-Analysis-PowerBI
Sales performance dashboard analyzing 244K revenue across products, categories, locations &amp; payment methods using Power BI
## 📊 Overview
An interactive Power BI dashboard analyzing Amazon sales performance 
across products, categories, customer locations, and payment methods.

## 🔍 Key Insights
- **Total Sales:** $244K | **Orders:** 250 | **Quantity Sold:** 714
- **Top Category:** Electronics leads with ~$140K in sales
- **Top Product:** Refrigerator is the highest-selling product (~$75K)
- **Top City:** Miami generates the highest revenue (~$32K)
- **Payment:** PayPal is most used (24% of orders)
- **Order Status:** 35.2% Completed | 34% Pending | 30.8% Cancelled
- **Trend:** Sales peaked in Feb 2025, sharp decline through April

## ⚠️ Business Problem
34% orders still pending and 30.8% cancelled — combined that's 65% 
of orders NOT completing. This is a critical fulfillment issue.

## 🛠️ Tools Used
- Power BI Desktop
- DAX (calculated KPIs)
- Excel/CSV (data source)
